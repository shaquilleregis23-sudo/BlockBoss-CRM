FAQ

Solar Canvassing CRM — Common Questions

What is a solar canvassing CRM?
A solar canvassing CRM is field sales software built for door-to-door solar teams. It replaces paper lists and spreadsheets with a map-based system that tracks every door knock, qualifies homeowners for solar, manages callbacks, and lets managers track rep activity in real time.
What is a Sun Score?
The Sun Score is BlockBoss CRM's proprietary lead priority algorithm. It scores each home 0–100 based on roof type, solar irradiance data for that ZIP code, homeowner status, property age, and existing solar installations nearby. Reps filter to high-score homes to maximize appointment rate.
How does BlockBoss compare to SalesRabbit for solar canvassing?
BlockBoss CRM includes NYC PLUTO homeowner data (real names before you knock), a Sun Score algorithm, full offline mode, and flat-rate pricing starting at $49/month — no per-seat fees. SalesRabbit is a solid general-purpose tool but doesn't include NYC-specific homeowner data or the Sun Score, and charges per user per month.
Can I track solar appointments set per rep?
Yes. BlockBoss CRM tracks every disposition per rep — including Appointment Set. The daily Manager Dashboard shows appts set today, this week, and all-time per rep. The Leaderboard ranks reps by doors knocked, appts set, and closed deals.
Does BlockBoss CRM work outside NYC for solar teams?
Yes. All core features — map canvassing, offline mode, rep tracking, leaderboard, callbacks — work anywhere. The NYC PLUTO homeowner data feature is specific to the five boroughs, Long Island, and surrounding areas, but teams outside NYC can import their own lead lists via CSV.
